JavaScript – Metode za Array in Object

Metode za Array
Metoda Opis Primer
.forEach() Izvede funkcijo za vsak element arr.forEach(x => ...)
.map() Ustvari nov array z rezultati funkcije arr.map(x => x * 2)
.filter() Vrne elemente, ki ustrezajo pogoju arr.filter(x => x > 5)
.reduce() Akumulira v eno vrednost arr.reduce((a, b) => a + b)
.find() Vrne prvi ujemajoči element arr.find(x => x > 5)
.some() Ali vsaj en ustreza pogoju? arr.some(x => x > 0)
.every() Ali vsi ustrezajo pogoju? arr.every(x => x > 0)
.includes() Ali vsebuje vrednost? arr.includes(3)
.sort() Uredi elemente arr.sort((a, b) => a - b)
.flat() Splošči ugnezdene arraye [1, [2]].flat()
.findIndex() Indeks prvega ujemanja arr.findIndex(x => x == 5)
Metode za Object
Metoda Opis Primer
Object.keys(obj) Array ključev ["ime", "starost"]
Object.values(obj) Array vrednosti ["Ana", 25]
Object.entries(obj) Array parov [ključ, vrednost] [["ime", "Ana"]]
Object.fromEntries() Array parov → objekt { a: 1, b: 2 }
obj.hasOwnProperty("ime") Ali obstaja ključ? true / false
